总结

共计优先级
21>nfs(111,2049,20048)>SAMBA(445)>80

  • NFS（Network File System，网络文件系统）是一种分布式文件系统协议，由 ​​Sun Microsystems​​ 开发，允许客户端像访问本地文件一样访问远程服务器上的文件。
  • 111端口由 rpcbind服务监听，NFS依赖RPC机制实现远程文件访问， rpcbind记录并管理所有RPC服务的动态端口，然后再通过RPC访问
