## base64加密
1. 首先建立一个0~63的数据字典
2. 然后将字符转为二进制
3. 原先按8个划分，现在6个位为一组重新分组
4. 对应分组对应base64编码表

## base64解密
1. 移除填充代码
```bash
原始字节数 = (Base64字符串长度 × 6) / 8 - 填充数
```
2. 字符反向映射
```bash
A→0, B→1, ..., Z→25
a→26, b→27, ..., z→51
0→52, 1→53, ..., 9→61
+→62, /→63
```
3. 二进制重组​
