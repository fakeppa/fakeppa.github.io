### 前言
首先注意版本，python版本必须为3.8才可以
## 1.反编译程序


解压附件获得something二进制程序
需要用到`pyinstxtractor`
```bash
# 安装并运行 pyinstxtractor
git clone https://github.com/extremecoders-re/pyinstxtractor
cd pyinstxtractor
python3 pyinstxtractor.py /path/to/something

# 进入解包目录
cd something_extracted
```
